# Security Sentinel Dashboard

Advanced Security & Vulnerability Scanner for Web, Mobile, and AI Models.

## Features
- **Web Scanning**: Acunetix integration for DAST.
- **Mobile Scanning**: MobSF integration for static and dynamic analysis.
- **AI Model Scanning**: Garak integration for LLM security probing.
- **Network Scanning**: Nmap integration for port discovery.
- **Unified Vulnerability Management**: Single dashboard for all findings.

## Tech Stack
- React + Vite + TypeScript
- Tailwind CSS + Shadcn UI
- Python FastAPI (Security Bridge)
- External Tools: Acunetix, MobSF, Garak, Nmap
